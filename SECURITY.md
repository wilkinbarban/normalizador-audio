# Security Policy

## Supported Versions

Security fixes are currently provided for:

- `1.x` (active)

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use one of these private channels:

- Open a private security advisory in GitHub: Security tab -> Advisories
- Email the maintainer: wilkinbarban@users.noreply.github.com

When reporting, include:

- Affected version
- Reproduction steps
- Expected vs actual behavior
- Logs or screenshots (without sensitive data)

## Response Targets

- Initial acknowledgement: within 48 hours
- First triage decision: within 5 business days
- Fix timeline: depends on severity and complexity

## Disclosure Policy

- We will coordinate disclosure after a fix is available.
- Credit will be given to reporters unless anonymity is requested.
